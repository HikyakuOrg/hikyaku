
**Hikyaku** is an open-source Logistics OS — a complete delivery management platform with a real-time dashboard for package tracking, fleet management, driver coordination, and route optimization. Built with Next.js, TypeScript, Tailwind CSS and Supabase. It integrates MapLibre GL and Valhalla for intelligent routing and live driver tracking. Features include multi-step package creation, shipping label generation (QR + barcode), VRP-based route optimization, and full warehouse management.

## Getting Started

1. Copy `.env.example` to `.env.local` and configure your environment variables.


2. Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
